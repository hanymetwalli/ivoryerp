# 🏢 Ivory ERP System - v4.32
نظام متكامل لإدارة الموارد البشرية (HRM) والاستقطاب والتوظيف (ATS) مصمم للعمل في بيئات الإنتاج الحقيقية.

## 🚀 المميزات التقنية المنجزة:
- **نظام ATS متكامل:** إدارة كاملة للوظائف، طلبات التوظيف، وتقييم المقابلات.
- **إدارة صلاحيات ديناميكية:** نظام (RBAC) مع خاصية (Data Scoping) لتقييد الوصول حسب الأقسام.
- **تكامل البيانات:** تحويل المرشحين إلى موظفين بضغطة زر مع نقل كامل الملفات والبيانات.
- **هيكلة الإنتاج:** دعم كامل لملفات البيئة (.env) وتأمين المسارات للعمل على دومين موحد.

## 🛠 التقنيات المستخدمة:
- **Frontend:** React.js + Vite + Tailwind CSS.
- **Backend:** PHP (PDO) API Architecture.
- **Database:** MySQL.
- **Server Support:** Apache (.htaccess optimized).

## 🛡️ ملاحظات أمنية:
- تم استبعاد كافة البيانات الحساسة وقواعد البيانات والمرفقات الشخصية من المستودع للحفاظ على خصوصية البيانات (Data Privacy).
- النظام مهيأ للعمل المباشر على سيرفر الإنتاج `ivoryerp.com`.
